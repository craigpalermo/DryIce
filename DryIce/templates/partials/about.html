Frequently Asked Questions

What is DryIce?

DryIce is a simple way to transfer files between computers.

How do I use DryIce?

Follow these simple steps:

  1. Select a file to upload using the Browse button.
  2. Click Upload to upload a file.
  3. Click a file's button to copy its link to your system clipboard.
  4. Share the link! Anyone with the link can download the file until it expires.

Who can download my files?

Only you can see the files that you've uploaded when visiting the site, but anyone with the link to one of your files can download it. There is no way for users to see a list of files uploaded by other users.

Can I share sensitive information through DryIce?

You can upload anything you want to DryIce, but be aware that anyone with the link to one of your file can download it.

How long are my files stored on DryIce?

Files uploaded to DryIce are stored for 60 minutes after they're uploaded before they are automatically deleted. Links to deleted files are no longer valid after that time.

Why doesn't DryIce have user accounts?

DryIce is meant to be a simple way to temporarily make files available online so you can share between computers. Anyone with a link to a file on DryIce can download the file, but users can only see links to the files that they've uploaded. That information is linked to your browser to eliminate the need for users to log in.
