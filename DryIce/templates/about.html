Frequently Asked Questions

What is DryIce?

DryIce is a simple way to transfer files between computers.

How do I use DryIce?

  1. Select a file to upload using the Browse button.
  2. Click Upload to upload a file.
  3. Share the link! Anyone with the link can download the file until it expires.

What is the maximum file size that I can upload?

There is currently no maximum file size that you can upload.

Who can download my files?

Only you can see the files that you've uploaded when visiting the site, but anyone with the link to one of your files can download it. There is no way for users to see a list of files uploaded by other users.

How long are my files stored on DryIce?

Files uploaded to DryIce are stored for up to 24 hours after they're uploaded before they are automatically deleted. Links to deleted files are no longer valid after that time.
